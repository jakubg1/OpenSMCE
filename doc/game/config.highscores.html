config/highscores.json

Contains default scoreboard settings. These are used when launching the game for the first time or when resetting the highscores manually.

  • The root entry.
    • size: How many entries should be stored in a single highscore list.
    • defaultScores: Contains default highscore entries. The length of this list should be the same as the value of size field.
      • A highscore entry.
        • name: A player name which should be displayed.
        • score: How much points the player has scored.
        • level: The level on which the player has ended their game.
