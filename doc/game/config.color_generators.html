config/color_generators/*.json

Defines a Color Generator. Color generators are used to determine which color should spawn next in the Shooter.

  • The root object.
    • type: Defines the type of this Color Generator. Available values are:
      1. "random" - The colors are generated randomly from the pool.
      2. "nearEnd" - The colors are generated based on which colors are in proximity of a Path's endpoint.
    • hasToExist: Required if type is random. Whether in order for a color to be generated it needs to already exist on the board.
    • selectChance: Required if type is nearEnd. The chance for a ball's color to be selected for the next generation, starting from the frontmost sphere on a randomly chosen path.
    • pathsInDangerOnly: Required if type is nearEnd. Whether only paths in danger should be taken into account when generating colors. If no path is in danger, a random path is selected as if this was set to false.
    • colors: A list of colors which can be ever generated by this generator.
      • A single sphere ID.
      One of the following:
    • fallback: The color to be generated when this generator fails.
    • fallback: Another Color Generator to be used when this generator fails.
    • colorsRemoveIfNonexistent: A list of colors which will be removed from the Shooter if no instances of that color are present on the board.
      • A single sphere ID.
